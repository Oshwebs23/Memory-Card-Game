# Memory Card Game (React & Van Gogh Edition) (working progress)

Welcome to the **Memory Card Game**, a fun, interactive browser game built with React, featuring gorgeous Van Gogh paintings as the cards! Challenge your memory while admiring the artistry of Vincent van Gogh.

## Features

- **React-powered gameplay** for a fast and responsive experience.
- **Van Gogh Paintings**: Enjoy classic masterpieces as the face of each card.
- **Dynamic card shuffling** each round for extra replay value.
- **Score tracking** to challenge yourself.

## How to Play

1. Start the game to see cards face-down on the screen.
2. Click on a card to reveal a Van Gogh painting.
3. Click a second card to try to find a match.
4. If the cards match, they stay revealed. If not, they flip back down.
5. Continue until all pairs are matched.

## Tech Stack

- **JavaScript**: Core game logic
- **CSS**: Custom styling & responsive layout
- **HTML**: Structure & markup 
- **React**: Component-based UI and state management

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Oshwebs23/Memory-Card-Game.git
   cd Memory-Card-Game
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
